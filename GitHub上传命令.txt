🚀 GitHub上传命令 (请替换YOUR_USERNAME和YOUR_REPO为您的实际信息)
================================================================

# 1. 连接到您的GitHub仓库
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# 2. 设置主分支
git branch -M main

# 3. 上传到GitHub
git push -u origin main

================================================================
示例 (请替换为您的实际仓库地址):
================================================================

git remote add origin https://github.com/zhanshen/linken-sphere-apple-browser.git
git branch -M main  
git push -u origin main

================================================================
上传完成后的步骤:
================================================================

1. 📊 查看构建状态
   - 点击仓库页面的 "Actions" 标签
   - 等待构建完成 (约10-15分钟)
   - 绿色✅表示成功

2. 📦 下载macOS程序
   方法1: 从Actions下载
   - 点击最新的构建任务
   - 在 "Artifacts" 部分下载 "LinkenSphereAppleBrowser-macOS"
   
   方法2: 从Releases下载  
   - 点击仓库页面的 "Releases" 标签
   - 下载最新版本的macOS文件

3. 🎉 获得所有平台程序
   - LinkenSphereAppleBrowser-Windows.exe (Windows版本)
   - LinkenSphereAppleBrowser-macOS (macOS版本) ← 您需要的！
   - LinkenSphereAppleBrowser-Linux (Linux版本)

================================================================
如果遇到问题:
================================================================

1. 确保仓库地址正确
2. 确保网络连接正常
3. 如果推送失败，可能需要先拉取: git pull origin main
4. 如果还有问题，可以删除仓库重新创建

================================================================
